# Close Monitor

### 使用
此应用将使用Pythonw.exe来启动
这意味着它将不会显示命令行窗口

将此py文件放在桌面并将打开方式设置为Pythonw.exe以双击启动该应用

双击后显示器将被关闭

### 注意
移动鼠标或其他操作将重新打开显示器
